# Erick Sixto — Personal Website

Portfolio and professional website for **Erick Sixto**, a Senior Salesforce Developer with 7+ years of experience building systems across Sales Cloud, Service Cloud, and Experience Cloud.

**Live site:** [ericksixto.github.io](https://ericksixto.github.io)

## Tech Stack

- **Framework:** React 19 with Create React App + CRACO
- **Styling:** Tailwind CSS + shadcn/ui components (Radix UI primitives)
- **Animations:** Framer Motion
- **Routing:** React Router DOM v7
- **Forms:** React Hook Form + Zod validation
- **Icons:** Lucide React

## Pages

| Route | Description |
|-------|-------------|
| `/` | Hero, services overview, featured projects, process, and social proof |
| `/about` | Background, experience timeline, and approach |
| `/services` | Full list of Salesforce services offered |
| `/projects` | Case studies with measurable business impact |
| `/contact` | Contact form and booking links |

## Development

```bash
# Install dependencies
yarn install

# Start dev server
yarn start

# Production build
yarn build
```

## Deployment

This site deploys automatically to GitHub Pages via GitHub Actions on every push to `main`.
